All about Cookies

Cookies are data, stored in small text files, on your computer. When a web server has sent a web page to a browser, the connection is shut down, and the server forgets everything about the user.

Creating Cookies

In JavaScript you write (and save) cookies like this:

document.cookie = "username=John Doe; expires=Thu, 18 Dec 2013 12:00:00 UTC";

Reading out Cookies

You can read out the cookies which are saved like this.

document.cookie;

Deleting Cookies

You don't have to specify a cookie value when you delete a cookie. Just set the expires parameter to a passed date:

document.cookie = "username=; expires=Thu, 01 Jan 1970 00:00:00 UTC; path=/;";
